## 1. 薬機法・医療広告

医療用医薬品・医療機器について一般向けに発信する際の最重要チェック項目です。

確認内容

対応策・記載例

- [ ] 
特定の薬剤・機器の効果を断定的に表現していないか

「○○は最も効果的」→「○○試験では△△が示された（出典）」に変更

- [ ] 
承認外の効能・効果を記載していないか

適応外使用の記載は「現時点では適応外」と明記

- [ ] 
有効性・安全性のデータに出典を明記しているか

「（Yam JC, et al. Ophthalmology. 2019）」などを必ず付記

- [ ] 
一般向け記事であることを明示しているか

冒頭に「医師・医療従事者向け」または「一般の方は主治医にご相談ください」を記載

- [ ] 
副作用情報の記載方法は適切か

「添付文書に記載の副作用として○○がある」など客観的表現を使う


## 2. COI（利益相反）開示

製薬会社・医療機器メーカーとの関係がある場合は必ず開示してください。

確認内容

対応策・記載例

- [ ] 
記事の内容に関連する企業との契約・報酬関係があるか

ある場合は末尾に「COI開示：○○社よりアドバイザリー報酬を受領」と明記

- [ ] 
講演料・研究費・顧問料を受領している企業の製品を肯定的に紹介していないか

受領関係がある場合は「個人的見解であり、企業の公式見解ではない」を併記

- [ ] 
所属機関の公式見解と混同されないか

「本記事は個人の見解であり、所属機関の立場を代表するものではありません」を記載

- [ ] 
スポンサードコンテンツの場合、明示しているか

「PR」「sponsored」「企業提供」などを冒頭に明示


## 3. 名誉毀損・プライバシー

確認内容

対応策・記載例

- [ ] 
特定の企業・製品・個人を根拠なく批判していないか

批判は「○○学会報告では」「○○データでは」など客観資料に基づく形にする

- [ ] 
患者情報・症例情報を掲載していないか

掲載する場合は匿名化＋情報マスキング＋本人同意取得を徹底

- [ ] 
Xの引用投稿：公開アカウントであることを確認したか

鍵アカウントの投稿は引用不可。一般患者の投稿は「Xユーザー」と匿名化

- [ ] 
他の医師・専門家の発言を正確に引用しているか

文脈を外した切り取りは名誉毀損リスク。引用元URLを明記

- [ ] 
実名医師のXを引用する場合、事前に一声かけたか

リプライで「note記事で引用してよいですか」と確認するとKOLとして印象が良い


## 4. 科学的正確性

確認内容

対応策・記載例

- [ ] 
データ・統計の出典を明記しているか

論文引用は「著者名, 誌名, 年」を最低限記載

- [ ] 
AIが生成した数値・情報をそのまま使っていないか

必ず一次文献で確認。AIの数値は誤りが含まれる場合がある

- [ ] 
エビデンスレベルを適切に示しているか

「パイロット試験では」「RCTで示された」など確実性のレベルを明示

- [ ] 
未確立の概念・造語を使う場合、明示しているか

「筆者の提唱する概念」「査読論文として確立されたものではない」と記載

- [ ] 
事後解析・サブグループ解析の限界を説明しているか

「事後解析のため参考値として解釈が必要」などを付記


## 5. 媒体別チェック

note（一般向け記事）

確認内容

対応策・記載例

- [ ] 
冒頭200字に主要キーワード（疾患名・薬剤名）が含まれているか

SEO効果のため疾患名・薬剤名をタイトルと冒頭に入れる

- [ ] 
タグは5個以内・具体的な疾患名・薬剤名を含めているか

例：#リジュセア #近視進行抑制 #眼科 #低濃度アトロピン #小児眼科

- [ ] 
著者名のハッシュタグ（#柳靖雄 #柳 靖雄 等）は入れていないか

著者名タグは使わない。テーマ・疾患・検査名など内容に即したタグのみ

- [ ] 
関連記事への内部リンクを末尾に入れているか

2〜3本のリンクで回遊率アップ

- [ ] 
免責事項を末尾に記載しているか

「本記事は情報提供を目的とし、診療の代替ではありません」
