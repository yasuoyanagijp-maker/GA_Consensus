# GA コンセンサス・リポート（ドラフト一式）

## ローカル編集 UI

`../../ga-consensus-editor/` でプレビュー編集・Zotero 文献表示が可能。起動: `cd ga-consensus-editor && npm install && npm run dev` → http://localhost:5173
